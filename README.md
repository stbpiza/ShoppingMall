# ShoppingMall


## 기능

- 카테고리 목록 확인
- 상품 목록 확인
- 상품 상세 정보 확인
- 로그인
- 로그아웃
- 회원 가입
- 장바구니에 상품 담기
- 장바구니 확인
- 주문하기 (배송지 입력, 결제)
- 주문 목록 확인
- 주문 상세 확인

## 관리자 기능

- 로그인
- 로그아웃
- 회원 관리
    - 회원 목록
- 카테고리 관리
    - 카테고리 목록
    - 카테고리 등록
    - 카테고리 수정
- 주문 관리
    - 주문 목록
    - 주문 상세
    - 주문 상태 변경
- 상품 관리
    - 상품 목록
    - 상품 상세
    - 상품 등록
    - 상품 수정
